# 五子棋游戏

## 游戏介绍
这是一个基于React和TypeScript开发的五子棋对战游戏，玩家可以与AI进行对战。

## 功能特性

### 🎮 游戏功能
- **15x15标准棋盘**：经典五子棋棋盘尺寸
- **智能AI对手**：使用评分算法实现智能AI
- **实时对战**：流畅的玩家与AI对战体验
- **胜负判定**：自动检测五子连珠获胜条件

### ⏰ 时间管理
- **总时间限制**：每局游戏10分钟总时间
- **单次出手限制**：每次出手30秒时间限制
- **实时倒计时**：显示剩余总时间和出手时间
- **超时处理**：超时自动判负

### 🎯 游戏控制
- **开始游戏**：开始新的对战
- **暂停/继续**：随时暂停或继续游戏
- **重新开始**：重置游戏状态
- **游戏状态显示**：清晰显示当前游戏状态

### 🎨 界面设计
- **现代化UI**：使用Ant Design组件库
- **响应式设计**：支持不同屏幕尺寸
- **动画效果**：棋子下落动画
- **视觉反馈**：悬停效果和状态提示

## 游戏规则
1. 黑子（玩家）先手，白子（AI）后手
2. 在15x15的棋盘上轮流下棋
3. 率先在横、竖、斜任意方向连成5子者获胜
4. 每局游戏总时间10分钟
5. 每次出手时间限制30秒

## 技术实现

### 前端技术栈
- **React 18**：现代化React框架
- **TypeScript**：类型安全的JavaScript
- **Ant Design**：企业级UI组件库
- **Less**：CSS预处理器

### AI算法
- **位置评估**：基于连子数量和阻挡情况评分
- **攻防策略**：优先阻止对手获胜，同时寻找自己的获胜机会
- **智能选择**：评估所有可能位置，选择最优下法

### 状态管理
- **游戏状态**：等待、进行中、暂停、结束
- **时间管理**：总时间和单次出手时间倒计时
- **棋盘状态**：15x15二维数组表示棋盘
- **回合管理**：玩家和AI轮流下棋

## 使用方法
1. 访问 `/game/gomoku` 路由进入游戏
2. 点击"开始游戏"开始对战
3. 在棋盘上点击空位下棋
4. 使用控制按钮管理游戏状态
5. 率先连成5子者获胜
