# 五子棋游戏使用说明

## 🎮 游戏介绍
我已经成功为你创建了一个完整的五子棋游戏，包含以下功能：

### ✨ 主要功能
1. **智能AI对战** - 使用评分算法实现智能AI对手
2. **时间管理** - 总时间10分钟，单次出手30秒限制
3. **实时倒计时** - 显示剩余总时间和出手时间
4. **游戏控制** - 开始、暂停、继续、重新开始
5. **现代化UI** - 使用Ant Design组件库，响应式设计

### 🎯 游戏规则
- 15x15标准棋盘
- 玩家（黑子）先手，AI（白子）后手
- 率先连成5子者获胜
- 总时间限制：10分钟
- 单次出手时间：30秒
- 悔棋次数：每局游戏最多3次

## 🚀 如何访问游戏

### 方法一：通过菜单导航
1. 启动项目：`npm run dev`
2. 在浏览器中访问：`http://localhost:5173`
3. 在左侧菜单中找到"游戏中心"
4. 点击"五子棋"进入游戏

### 方法二：直接访问
直接访问：`http://localhost:5173/game/gomoku`

## 🎮 游戏操作

### 开始游戏
1. 点击"开始游戏"按钮
2. 游戏开始，总时间开始倒计时
3. 玩家先手，在棋盘上点击空位下棋

### 游戏控制
- **开始游戏** - 开始新的对战
- **暂停/继续** - 暂停或继续当前游戏
- **悔棋功能** - 每局游戏可悔棋3次，撤销玩家和AI的最后一步
- **重新开始** - 重置游戏状态

### 悔棋功能详解
- **悔棋次数** - 每局游戏最多3次悔棋机会
- **悔棋范围** - 每次悔棋会撤销玩家和AI的最后一步
- **悔棋条件** - 需要至少下过2步才能悔棋
- **悔棋显示** - 按钮显示剩余悔棋次数，用完后按钮变灰
- **状态恢复** - 悔棋后会恢复到悔棋前的游戏状态和时间

### 时间管理
- **总时间** - 显示剩余游戏总时间（红色警告：<60秒）
- **出手时间** - 显示当前回合剩余时间（红色警告：<10秒）
- **超时处理** - 超时自动判负

## 🧠 AI智能程度

### 算法特点
- **攻防兼备** - 优先阻止对手获胜，同时寻找自己的获胜机会
- **位置评估** - 基于连子数量和阻挡情况智能评分
- **多步思考** - 考虑不同方向的连子可能性
- **智能选择** - 从所有可能位置中选择最优下法

### 难度等级
当前AI为中等难度，适合大多数玩家挑战

## 🎯 游戏技巧

### 对玩家建议
1. **先手优势** - 利用先手优势占据中心位置
2. **防守优先** - 注意阻止AI形成四子连线
3. **时间管理** - 合理利用30秒出手时间
4. **观察全局** - 不要只看局部，要观察整个棋盘

### 获胜策略
1. 优先形成自己的四子连线
2. 阻止对手形成威胁
3. 控制棋盘中心区域
4. 保持冷静，不要急于求成
